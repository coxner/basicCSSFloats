Project Takeaway

One of the tricky parts of CSS I took away from this project is the clearfix hack. Float was designed to align images with text so using it to display content requires a few workarounds.


The simpliest way to define the clearfix is that it will display the columns as type block and allow the div to have a height. Without using the clearfix you will experience problems with displaying your content.
